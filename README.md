# Marketing Performance Snapshot Dashboard

## Project Overview

This project was created as part of the Data & AI Product Engineer assessment.

The goal of the project is to design a lightweight internal dashboard that helps marketing teams quickly understand campaign performance across multiple channels without manually checking different platforms.

The dashboard focuses on:
- Cross-channel marketing analysis
- Campaign performance monitoring
- Audience insights
- Engagement tracking
- Decision-support reporting

---

## Dashboard Summary

The dashboard provides:
- Overall marketing performance KPIs
- ROI comparison across channels
- Monthly engagement trends
- Campaign conversion analysis
- Audience and customer segment insights

The solution is intentionally designed to be simple, focused, and easy to use.

---

## Tools Used

- Microsoft Power BI
- CSV Dataset from Kaggle
- Draw.io for workflow diagram
- GitHub for version control and documentation

---

## Key Insights Generated

- Email and YouTube campaigns generated the highest ROI.
- Influencer campaigns achieved the highest conversions.
- Customer engagement varied significantly across customer segments.
- Monthly click trends helped identify seasonal engagement changes.
- Some channels showed lower engagement despite high visibility.

---

## V1 Scope

The first version focuses only on:
- Marketing performance overview
- Cross-channel comparison
- Campaign monitoring
- Audience insights
- Exportable reporting

The solution prioritizes simplicity and usability over advanced analytics.

---

## Out of Scope

The following features are intentionally excluded:
- AI-generated recommendations
- Predictive forecasting
- Real-time streaming analytics
- Automated campaign optimization
- Custom client dashboards
